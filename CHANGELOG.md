# Changelog

## v0.1.0

Initial release.

- ProjectE EMC integration for Magitech -Arcane Engineering- (NeoForge 1.21.1).
- Hand-set EMC for Magitech's root resources (ores, gems, mana resources, worldgen plants, crystals).
- Magitech's Zardius crucible / spell conversion / athanor recipes transcribed as ProjectE conversions so element crystals, special ingots and alchemy products derive automatically.
- Tier-4 endgame materials priced high to keep mining preferable to transmutation.
- Tool / part system intentionally left without EMC (material-parametric).
- Data-only: adds no items, blocks or recipes.
